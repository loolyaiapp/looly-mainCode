🎤  "Can you explain the difference between process and thread, and when would you use one over the other?"
Streaming answer

A process is an independent program in execution with its own memory space. A thread is a lightweight unit within a process that shares memory.

Use threads when tasks need to share data frequently (e.g. a web server handling requests). Use processes for isolation — if one crashes, others survive (e.g. browser tabs).
